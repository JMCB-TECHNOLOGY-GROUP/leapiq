# LeapIQ

An adaptive AI educational platform that personalizes learning from Pre-K through College. LeapIQ uses Anthropic Claude to generate curriculum-aligned lessons, quizzes, and assessments that adapt in real time to each student's performance level and learning pace.

Built by [JMCB Technology Group](https://jmcbtechgroup.com).

## Key Features

- **Adaptive Learning Engine** -- Spaced repetition, difficulty calibration, and knowledge gap targeting that adjusts to each student
- **AI-Generated Questions** -- Standards-aligned questions generated by Claude, mapped to Bloom's Taxonomy levels and state curriculum standards
- **AI Tutor** -- Conversational tutoring with pedagogically-grounded responses, growth-mindset language, and age-appropriate explanations
- **Learning Gap Analysis** -- AI-powered identification of weak areas with personalized remediation learning paths
- **Document-to-Quiz** -- Upload study materials (text or images) and generate quiz questions automatically
- **Multi-Role Dashboards** -- Separate views for Students, Educators, and Parents
- **Progress Visualization** -- Recharts-powered charts showing accuracy, XP, and learning velocity over time
- **50-State Standards** -- Questions aligned to each state's educational standards (Common Core, NGSS, state-specific)
- **Grade Range Coverage** -- Pre-K through College with age-appropriate content and vocabulary

## Quick Start

### Prerequisites

- Node.js 20+
- npm 10+
- Anthropic API key

### Setup

```bash
# Clone the repository
git clone https://github.com/jmcbtechgroup/leapiq.git
cd leapiq

# Install dependencies
npm install

# Create environment file
cp .env.example .env.local
# Add your ANTHROPIC_API_KEY

# Start the development server
npm run dev
```

### Environment Variables

```env
ANTHROPIC_API_KEY=your_anthropic_api_key
```

Open [http://localhost:3000](http://localhost:3000) to access the platform.

## Architecture Overview

LeapIQ is a single-page application with AI-powered API routes. All student data is currently stored in localStorage with a planned migration to Supabase.

```
src/
  app/                    # Root layout and landing page
  app/api/
    analyze-gaps/         # AI learning gap analysis
    chat/                 # AI tutor conversation
    generate-questions/   # AI quiz question generation
    generate-from-doc/    # Document-to-quiz conversion
  components/
    DocUpload.tsx         # Study material upload
    EducatorDashboard.tsx # Teacher view with classroom management
    EducatorLogin.tsx     # Educator authentication
    GenQuizPractice.tsx   # AI-generated quiz practice mode
    Landing.tsx           # Home page with role selection
    ParentDashboard.tsx   # Parent progress monitoring view
    ParentLogin.tsx       # Parent authentication
    ProgressCharts.tsx    # Recharts visualizations
    Quiz.tsx              # Core quiz-taking component
    ReviewMode.tsx        # Spaced repetition review
    StudentDashboard.tsx  # Student home with subjects and stats
    StudentLogin.tsx      # Student profile creation
    Tutor.tsx             # AI chat tutor interface
  data/
    questions.ts          # Static question bank (fallback)
  lib/
    adaptive-engine.ts    # Core adaptive learning algorithm
    app-context.tsx       # React context for global app state
    constants.ts          # Grade levels, subjects, US states
    storage.ts            # localStorage persistence layer
    types.ts              # TypeScript interfaces
```

### Core Algorithm

The adaptive engine in `lib/adaptive-engine.ts` combines three strategies:

1. **Spaced Repetition** -- Leitner box system tracks mastery per question; due items are prioritized
2. **Difficulty Calibration** -- Questions sorted by difficulty for progressive challenge
3. **Gap Targeting** -- Categories below 70% accuracy are flagged for focused practice
